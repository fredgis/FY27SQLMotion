# SQL Motion — Brief

> Detailed narrative brief behind the one-slide summary. The slide is the
> executive view; this document is the working detail.

## Objective

Drive **Azure SQL** through two complementary go-to-market paths — **migration**
delivered by System Integrators, and **adoption** by ISVs — enabled by a single
hands-on vehicle, **"SQL in a Day."**

## Path 1 — Migration / System Integrators

- **Motion:** Migrate SQL workloads to **Azure**, delivered through **GSIs**.
- **Approach:** Migration via GSI.
- **KPI:** **Migrate** — number of opportunities (Nbr Opty).
- **GSI partners:** Capgemini, Accenture / Avanade, NTT Data, TCS, IBM (Nordcloud).

## Path 2 — Adoption / ISV (SDC)

- **Motion:** ISVs **adopt a managed database service** and operate on Azure SQL.
- **Approach:** Adoption / Operate.
- **KPI:** **Operate** — % of ISVs that adopted a new service.
- **Goal:** Each ISV runs a new managed database service on Azure.

## Enablement vehicle

- ⚡ **SQL in a Day** — e.g. help CSPs build their own "SQL in a Day."

## The work

- List all technical migration assets.
- List all prerequisites to accelerate migrations.
- Define the rationale — *why migrate?* (PG "new born" · SQL Server "new born").

## The ask

- Funding.
- 8 meetings — lead STU.
- 1 sponsor per GSI.
